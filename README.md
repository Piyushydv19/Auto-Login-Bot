# 🚀 AutoLogin Bot

> Intelligent multi-site login automation platform built using Flask + Selenium.

AutoLogin Bot is a browser automation platform that allows users to manage and log into multiple websites through a modern dashboard interface. The project combines intelligent Selenium automation with a clean frontend UI to simplify authentication workflows and multi-account management.

## 📸 Dashboard Preview

![AutoLogin Bot Dashboard](screenshots/dashboard.png)
## 💡 Why This Project?

Managing multiple website logins manually can be repetitive and inefficient.
AutoLogin Bot automates authentication workflows through intelligent browser automation and multi-profile management.
---

## ✨ Features

* 🚀 One-click login execution
* 🔐 Multi-site login automation
* 👤 Multiple profiles for the same website
* ➕ Add and manage accounts directly from dashboard
* 🧠 Automatic login field detection
* ⚡ Human-like typing and clicking simulation
* 📊 Real-time login status tracking

* 🔄 Frontend + backend synchronization

## 🚀 Getting Started

### 1. Clone Repository

```bash
git clone https://github.com/your-username/AutoLogin-Bot.git
cd AutoLogin-Bot
```

### 2. Install Dependencies

```bash 
pip install selenium flask flask-cors
```

### 3. Run Project

```bash
python app.py
```

Open:

```bash
http://127.0.0.1:5000
```

---

## ⚙️ How It Works

1. Add website credentials from the dashboard
2. Multiple profiles can be created for the same website
3. Data automatically syncs with `credentials.json`
4. Selenium launches browser automation
5. Login fields are detected automatically
6. Login process executes with live status updates
